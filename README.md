# 📝 To-Do List App

An intuitive, interactive web application to manage your daily tasks efficiently, built using Python and Streamlit.

## ✨ Features

* **Add Tasks:** Quickly add new items to your daily checklist.
* **Track Progress:** Mark tasks as completed with interactive checkboxes.
* **Clean Interface:** Minimalist and responsive UI designed with Streamlit.
* **Instant Deployment:** Hosted on Hugging Face Spaces for seamless access.
